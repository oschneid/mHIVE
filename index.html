mHIVE

mHIVE is a mobile Haptic Instrument for Vibrotactile Exploration, an Android application that enables rapid prototyping of haptic feedback. Part of the Haptic Jazz project, mHIVE takes inspiration from musical instruments, providing real-time feedback for fast iteration, displaying to multiple outputs to facilitate feedback.
